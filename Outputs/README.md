# Outputs

Generated samples, renderer smoke outputs, and local run artifacts belong here.

Default roots:

- `Outputs/Samples/` for template-generated supervised samples.
- `Outputs/PaintTransformerSamples/` for Paint Transformer-backed supervised samples.
- `Outputs/RenderSmoke/` for local renderer smoke runs.
- `Outputs/ArtInstituteChicago/` for Art Institute of Chicago import and border-removal manifests plus related local metadata.
- `Outputs/Openclipart/` for Openclipart import manifests and related local metadata.
- `Outputs/GoogleQuickDraw/` for Google Quick, Draw! import manifests and CC-BY-4.0 attribution metadata.
- `Outputs/ArtBench/` for ArtBench import manifests and related local metadata.

This directory is ignored by Git except for this README. Tests should prefer temporary directories instead of writing here.
